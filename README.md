# Supermarket App 🛒

> [cite_start]Μια Android εφαρμογή για τη διαχείριση αγορών από σούπερ μάρκετ[cite: 20]. [cite_start]Επιτρέπει την εύκολη περιήγηση σε προϊόντα, τη δημιουργία λιστών αγορών, την αποθήκευση σε λίστα επιθυμιών και την παρακολούθηση του ιστορικού παραγγελιών [cite: 21][cite_start], με πλήρη υποστήριξη διγλωσσίας (Ελληνικά/Αγγλικά)[cite: 22, 36].

## ✨ Βασικές Λειτουργίες (Features)
* [cite_start]**Κατάλογος Προϊόντων:** Περιήγηση σε 30 προϊόντα από 10 διαφορετικές κατηγορίες, με δυνατότητα αναζήτησης και φιλτραρίσματος (π.χ. προβολή μόνο προσφορών)[cite: 27, 28].
* [cite_start]**Λεπτομέρειες Προϊόντος:** Αναλυτική προβολή πληροφοριών, όπως θρεπτικά στοιχεία, τιμή, διαθεσιμότητα και εκπτώσεις[cite: 29].
* [cite_start]**Λίστα Αγορών & Checkout:** Προσθήκη προϊόντων στο καλάθι, ρύθμιση ποσοτήτων και υπολογισμός συνολικού κόστους σε πραγματικό χρόνο[cite: 30].
* [cite_start]**Λίστα Ευχών (Wish List):** Αποθήκευση προϊόντων για μελλοντικές αγορές με εύκολη μεταφορά τους στο καλάθι[cite: 31].
* [cite_start]**Ιστορικό Αγορών:** Καταγραφή προηγούμενων παραγγελιών με δυνατότητα άμεσης επανάληψης[cite: 32].
* [cite_start]**Τοπικοποίηση:** Πλήρης υποστήριξη Ελληνικών και Αγγλικών σε όλο το UI και τα δεδομένα[cite: 36, 37, 39].

## 🛠️ Τεχνολογίες & Αρχιτεκτονική (Tech Stack)
* [cite_start]**Γλώσσα:** Kotlin (έκδοση 2.0.21) [cite: 42]
* [cite_start]**Αρχιτεκτονική:** MVVM (Model-View-ViewModel) [cite: 48]
* [cite_start]**UI Framework:** Jetpack Compose με σχεδιασμό Material 3 [cite: 43]
* [cite_start]**Πλοήγηση:** Navigation Compose [cite: 46]
* [cite_start]**Τοπική Αποθήκευση:** Room Database [cite: 44]
* [cite_start]**Φόρτωση Εικόνων:** Coil [cite: 45]
* [cite_start]**Διαχείριση Δεδομένων:** Kotlinx Serialization για επεξεργασία JSON [cite: 46]
